### Auditoria Fiscal de NCM com IA
Ferramenta de auditoria e padronização de NCM para cadastros de produtos em sistemas ERP, utilizando **Python, Streamlit e IA generativa**. O projeto automatiza a detecção de inconsistências fiscais e gera relatórios de divergência, apoiando análises tributárias e estudos para transição de regimes (Simples → Lucro Real).

**Tecnologias:** Python, Streamlit, Pandas, OpenRouter API, OpenPyXL, Processamento Paralelo  
**Repositório:** [GitHub](https://github.com/rogeriobelo90/auditoria-fiscal-ncm-ia)
